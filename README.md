# ZhStudio

React/Vite website for ZhStudio. The current content is focused on Instagram and TikTok marketing for local businesses.

## Requirements

- Node.js
- npm

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Then open the local URL shown in the terminal, usually:

```text
http://localhost:5173
```

## Production Build

```bash
npm run build
```

The built files are generated in `dist/`.

## Main Files

- `src/App.jsx` - page content, routing, sections, and animations
- `src/styles.css` - complete visual styling and responsive layout
- `public/` - public assets and static files
- `index.html` - metadata, fonts, app mount
